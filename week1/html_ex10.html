TCP
Transmission Control Protocol is a method (Protocol) used along with the Internet Protocol to send data in the form of message units, called packets, between computers over the Internet
IP
Internet Protocol is the method or protocol by which data is sent from one computer to another on the Internet. Each computer on the Internet is unqiuely identified by an IP Address
